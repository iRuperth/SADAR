# SADAR — Hugging Face Space card

This file is the README used **only** when the project is deployed to a Hugging Face
Space. The YAML front-matter at the top is the Space card that Hugging Face needs to
configure the container (SDK, port, title, colors, license).

When deploying to Hugging Face Spaces, copy this file to the repository root as
`README.md` (overwriting the GitHub one) before pushing to the Space remote. The GitHub
`README.md` does not include this front-matter on purpose, so it renders cleanly on
github.com.

For the project description, screenshots, architecture and run instructions, see the
main `README.md` at the repository root.
